DSL Layout Input

DSL Format Rules:

  • Indentation-based structure (gunakan spasi atau tab)
  • Elemen didefinisikan dengan: type name props?
  • Tiga dash (---) menandai akhir dari props
  • Baris setelah --- adalah children (harus lebih indent)
  • Format props: key: value atau key value
  • Array values: at 0 0 menjadi "at": [0, 0]

Contoh DSL Layout:

@layout
normal
    at 0 0
    size 280 354
    ---
    Minimize: button at -41 4 anchor right
        =CLOSE()
